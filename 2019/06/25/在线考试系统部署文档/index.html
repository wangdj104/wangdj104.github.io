环境准备

开发环境：

  • jdk：1.8
  • MySQL：5.7+
  • redis
  • rabbitMq
  • node.js
  • consul
  • fastDfs（推荐基于docker安装，安装步骤网上很多）

开发工具：IntelliJ IDEA、WebStorm
